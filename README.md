# Notifyback

This is a Spring Boot project for TCC (Demo project).

## Requirements

- Java 17
- Maven 3.6+

## Building the Project

To build the project, run:

```bash
mvn clean install
```

## Running the Application

To run the application, use:

```bash
mvn spring-boot:run
```

The application will start on port 8080.
